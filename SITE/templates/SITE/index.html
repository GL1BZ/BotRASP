Удобство

Всё находится в одном месте!

Вы с лёгкостью сможете посмотреть всю интересующую вас информацию о расписании в привычном вам приложении. И чтобы это осуществить понадобится только подписаться на меня.

API

API для ваших целей

Чтобы получить информацию по расписанию, отправьте GET запрос на этот URL с указанием всех необходимых параметров:

  • searchName - Имя поиска(например: ИСиП-42)
  • dateNum - Дата поиска(например: 2023-02-22)
  • dataType - Вариант ответа JSON или фото
                                        
https://botrasp.ru/api/?searchName= Параметр поиска &dateNum= Дата &dataType= photo(или json)

GET запрос

Ниже представленны кусочки кода, которые можно сразу вставить в Ваш проект. В случае с сохранением изображения следует указывать директорию для сохранения файла (все изображения сохраняются в формате PNG) В некоторых случаях JSON массив может отображаться некорретно

--header 'Content-Type: application/json'
